## McKinsey SCQA Format

All strategic documents follow the SCQA (Situation, Complication, Question, Answer) framework.

## 8-Section Document Structure

### Section 1: Executive Summary (SCQA)

```
SITUATION
[2-3 sentences: Current state facts, context]

COMPLICATION  
[2-3 sentences: What changed, what threatens, the tension]

QUESTION
[1 sentence: The strategic question we're answering]

ANSWER
[1-2 sentences: The recommendation, bottom line up front]
```

### Section 2: Situation Analysis

- Market context and trends
- Competitive landscape overview
- Internal capabilities assessment
- Relevant data points

### Section 3: Problem Diagnosis

- Root cause analysis
- Assumption validation
- Impact quantification
- What's really driving this problem

### Section 4: Strategic Options

Present exactly 3 alternatives:

| Criteria | Option A | Option B | Option C |
|----------|----------|----------|----------|
| Description | [Name + summary] | [Name + summary] | [Name + summary] |
| Pros | [2-3 bullets] | [2-3 bullets] | [2-3 bullets] |
| Cons | [2-3 bullets] | [2-3 bullets] | [2-3 bullets] |
| Resources | [Cost, time, people] | [Cost, time, people] | [Cost, time, people] |
| Risk Level | [Low/Med/High] | [Low/Med/High] | [Low/Med/High] |

### Section 5: Recommendation

- Clear, single direction
- Rationale for selection (why this option)
- Expected outcomes with specific metrics
- What success looks like

### Section 6: Implementation Roadmap (30-60-90 Day)

| Timeframe | Actions | Owner | KPIs | Milestones |
|-----------|---------|-------|------|------------|
| Days 1-30 | Quick wins, foundation setting | [Role] | [Metrics] | [Deliverables] |
| Days 31-60 | Build momentum, core execution | [Role] | [Metrics] | [Deliverables] |
| Days 61-90 | Scale, optimize, measure | [Role] | [Metrics] | [Deliverables] |

### Section 7: Risk Mitigation

| Risk | Probability | Impact | Mitigation Strategy | Early Warning Signal |
|------|-------------|--------|---------------------|---------------------|
| [Risk 1] | High/Med/Low | High/Med/Low | [Action] | [What to watch] |
| [Risk 2] | High/Med/Low | High/Med/Low | [Action] | [What to watch] |
| [Risk 3] | High/Med/Low | High/Med/Low | [Action] | [What to watch] |

### Section 8: Alchemy Section (Premium)

Counterintuitive options based on behavioral economics:

**Counterintuitive Options**
- [2-3 alternatives that challenge conventional thinking]

**The Perception Play**
- [How to change how this FEELS rather than what it IS]

**Small Bet, Big Signal**
- [Low-cost (<$10K), high-impact intervention]

**The Hidden Driver**
- [The psychological reality beneath the stated problem]

## Quality Standards: SMART+O

Every recommendation must be:

- **S**pecific: Concrete action, not vague direction
- **M**easurable: Quantifiable success criteria  
- **A**ctionable: Can start within 48 hours
- **R**esourced: Budget, people, tools identified
- **T**ime-bound: Deadline specified
- **O**wned: Single accountable person/role named

## Formatting Guidelines

- Use tables for comparisons
- Use bullet points sparingly, prefer prose
- Lead with the conclusion, support with evidence
- Keep executive summary to one page
- Total document: 10-20 pages depending on complexity

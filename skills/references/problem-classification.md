# QEP AISolve — Problem Classification System

## 4-Layer Taxonomy

### Layer 1: Symptoms (Surface-Level User Language)

| Category | Example User Language |
|----------|----------------------|
| **Growth** | "Revenue is flat", "Not growing", "Lost market share" |
| **Efficiency** | "Costs are too high", "Taking too long", "Can't keep up" |
| **People** | "Can't hire fast enough", "High turnover", "Team underperforming" |
| **Market** | "Competitors winning", "Customers leaving", "Not differentiated" |
| **Financial** | "Not profitable", "Cash flow issues", "Margins shrinking" |

### Layer 2: Challenges (Underlying Business Issues)

| Symptom Category | Possible Challenges |
|------------------|---------------------|
| Growth | Channel inefficiency, targeting mismatch, value prop unclear, market saturation |
| Efficiency | Process bottlenecks, tech debt, resource misallocation, scale disadvantages |
| People | Employer brand weak, compensation misaligned, culture issues, management gaps |
| Market | Differentiation gap, speed disadvantage, customer insight gaps |
| Financial | Pricing misalignment, cost structure issues, capital inefficiency |

### Layer 3: Domains

| Domain | Primary Focus |
|--------|---------------|
| **Strategy** | Competitive positioning, growth direction, business models |
| **Marketing** | Demand generation, brand, positioning, channels |
| **Sales** | Pipeline, conversion, team performance, pricing |
| **Operations** | Processes, efficiency, scaling, delivery |
| **Innovation** | New products, R&D, disruption response |
| **HR** | Talent, culture, organization design |
| **Finance** | Profitability, cash flow, funding, metrics |

### Layer 4: Intent

| Intent | User Goal | Output Focus |
|--------|-----------|--------------|
| **Explore** | Understand options, gather information | Analysis-heavy, multiple perspectives |
| **Decide** | Choose between alternatives | Recommendation-focused, comparison |
| **Execute** | Implement a chosen path | Action-heavy, detailed roadmap |
| **Monitor** | Track progress, course correct | Metrics-focused, dashboards |

## Classification Examples

### Example 1
**User says**: "Our marketing spend is up 40% but leads are flat"
- **Symptom**: Growth (leads flat despite investment)
- **Challenge**: Channel inefficiency, targeting mismatch
- **Domain**: Marketing → Sales
- **Intent**: Decide

### Example 2
**User says**: "We can't hire engineers fast enough to ship features"
- **Symptom**: People (can't hire)
- **Challenge**: Employer brand, compensation, process
- **Domain**: HR → Operations → Innovation
- **Intent**: Execute

### Example 3
**User says**: "A competitor just launched a free tier. What do we do?"
- **Symptom**: Market (competitor threat)
- **Challenge**: Competitive response, differentiation
- **Domain**: Strategy → Marketing → Sales
- **Intent**: Decide
